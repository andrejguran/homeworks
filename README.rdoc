== Portal for automatic correction of homework

Rails application for automatic correction of homeworks
supports:
  java
  xquery
  xslt

== Instalation

install rails

  gem install rails

install dependencies

  bundle install

migrate and seed db

  rake db:reset

== Usage

  uco: 111111
